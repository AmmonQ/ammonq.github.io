Lecture Prep Website

I developed a new, updated Lecture Prep website as part of my duties as a Web Developer for the Brigham Young University Department of Chemistry and Biochemistry IT office. This website allows chemistry faculty to order chemical demonstrations in an efficient way. It also provides an organized way for the Lecture Prep staff to manage orders from faculty.

I used Django, python, HTML, CSS, JavaScript, and PyCharm to develop the website on a PostgreSQL database and GitLab for version control and OpenShift to deploy the website.
